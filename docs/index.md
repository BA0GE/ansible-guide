# Ansible 系统化指南

欢迎来到 **Ansible 系统化指南**。本文档旨在提供一份具备生产级参考价值的 Ansible 自动化深度参考手册。

## 项目总览

Ansible 是一款 **无代理 (Agentless)** 的自动化运维引擎，它利用 **SSH** 协议和 **幂等性 (Idempotency)** 机制，将复杂的配置管理、应用部署和任务编排过程转化为可读性极高的 **YAML** 剧本 (Playbooks)。
